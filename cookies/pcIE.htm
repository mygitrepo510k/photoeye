Here is an example of the Internet Explorer screen on your Windows machine:


Step1: Click on the Tools tab (upper left of your screen, circled in red above) and go to Internet Options (circled in red below.).


Step 2: Below is an example of the Internet Options window when it opens. Click on the Security tab (circled in red below).



Step 3: Below is an example with the Security tab activated (brought forward). Click on Custom Level, circled in red. A new window will open, described in Step 4 below.


Step 4: Below is an example with the Security Settings window opened. Scroll down, about 1/3 of the way down to Cookies, circled in red. Make sure that both Enable buttons are selected (where the red arrows point). Now click OK, circled in green. You will be asked "Are you sure you want to change the security settings for this zone?". Click Yes. Then click OK on the Internet Options window.
