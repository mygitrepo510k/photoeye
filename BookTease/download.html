Interactive 3D BookTease™ requires a Metastream plug-in to be installed in your Netscape or Microsoft Internet Explorer browser. You may download this small program for free from Metacreation's Metastream web site. There you will find instructions as to how to download and install it. Before you begin, please note the system and browser requirements.

After downloading the program, some browsers require that you manually install it. Please make sure you note the name of the directory into which you download the program. Then after it has completed the download process, click on the new program file on your computer. In case you can't find the file after it is downloaded, just do a search for a file named MetaStream*.exe.
